# Latent Studio – GitHub Pages

Statische Projektdokumentation ohne Build-Schritt.

## Veröffentlichen

1. Den Inhalt dieses Ordners in das Root-Verzeichnis eines GitHub-Repositories hochladen.
2. Unter **Settings → Pages** als Quelle **Deploy from a branch** wählen.
3. Branch `main` und Ordner `/ (root)` auswählen.
4. Die erzeugte GitHub-Pages-URL öffnen.

Die Seite ist responsiv und für den Browser-Export als PDF optimiert.
